What is an Exoplanet? Planets beyond our Solar System

An exoplanet, or extrasolar planet, is a planet that orbits a star outside our Solar System. The first evidence of an exoplanet was noted in 1917, but the first confirmation of its detection occurred only in 1992.

Since then, thousands of exoplanets have been discovered, ranging in size from gas giants larger than Jupiter to rocky planets smaller than Earth. These distant worlds are found through various methods, the most common being the transit method, where a telescope measures the slight dimming in a star's brightness as a planet passes in front of it. This is exactly the type of data our project uses.

What is Machine Learning? How computers learn from data

Machine Learning is a field of artificial intelligence that gives computers the ability to "learn" from data without being explicitly programmed. Instead of following a fixed set of rules, a machine learning model identifies patterns in training data to make predictions or decisions about new data.

In this project, we use machine learning for a classification task. We "train" our model with thousands of examples from NASA's exoplanet catalog, where each example has already been classified by scientists as "CONFIRMED" or "FALSE POSITIVE". The model learns to associate the features of a signal with a final classification.
